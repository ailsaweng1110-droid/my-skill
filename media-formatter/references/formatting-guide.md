## 三、标签体系（三类必须视觉区分，禁止混用）

### 二级序号标签
- **用途：** 章节编号 01/02/03
- 无容器，无背景
- 文字色：#0251FF，无边框，无圆角
- font-style: italic

### STEP 标签
- **用途：** 流程步骤
- 形状：实心胶囊
- 背景：#0251FF / 文字：#FFFFFF
- 无边框，圆角：12px（药丸形）
- 高度：24px / 内边距：0 12px

### 前后对比标签（AI优化前/AI优化后）
- **用途：** 内容前后对比
- 形状：空心方框
- 背景：#FFFFFF / 文字：#0251FF
- 边框：1.5px solid #0251FF，圆角：3px

### 灰色标签（优化前）
- **用途：** 标注原始/待优化内容
- 形状：实心方块
- 背景：#F0F0F0 / 文字：#999999
- 无边框，圆角：3px

### 所有标签统一参数
字号：11px / 字重：600 / 内边距：2px 8px / 字间距：1px

## 四、强调与高亮

### 行内高亮
- 字重：700（加粗）
- 底部色带：`linear-gradient(to top, rgba(2,81,255,0.12) 30%, transparent 30%)`
- 文字颜色保持 #222222 不变

### 品牌色文字
- 颜色：#0251FF / 字重：600
- 用于序号等需要强调但不需要底色的场景

## 十、设计原则

| 原则 | 说明 |
|-----|-----|
| 扁平克制 | 不使用渐变色块、阴影、圆角大于12px的元素 |
| 三色为主 | 白底#FFF + 正文#222 + 品牌蓝#0251FF，辅以灰色层级 |
| 无表情包 | 不使用meme图片、emoji表情符号 |
| 层级清晰 | 通过字号、字重、颜色三维度区分信息层级，避免依赖装饰元素 |
| 标签互斥 | 三类标签（斜体数字/实心胶囊/空心方框）形状完全不同，禁止混用 |
