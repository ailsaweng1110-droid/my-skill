# media-formatter · 公众号排版 Skill

> 负责排版输出阶段，将定稿内容转为可发布的 HTML。内容撰写请使用 → [`media-content-writer`](../media-content-writer/README.md)

---

## 安装方式

**前置条件**：已在龙虾 Agent 平台创建或进入目标 Agent，且 `media-content-writer` Skill 已安装完成。

**第一步：上传文件**
将以下文件按原有目录结构上传至 Agent 的 Skill 管理区：

```
media-formatter/
├── SKILL.md
└── references/
    └── formatting-guide.md
```

**第二步：配置 Skill**
在 Agent 平台的 Skill 设置中，将 `SKILL.md` 指定为该 Skill 的主入口文件，`references/` 文件夹设为关联资源目录。

**第三步：验证触发**
在对话框输入以下任意一句话，确认 Skill 正常触发：

```
内容写好了，帮我排版
```

Agent 应询问定稿正文和选定标题，说明 Skill 已生效。

> 具体操作路径因平台版本而异，如界面有差异请参考龙虾官方文档。

---

## 这个 Skill 是做什么的

接收已审阅定稿的文章正文，按「易点设计」品牌视觉规范，输出可直接粘贴进微信公众号编辑后台的 HTML 版本。

**不包含**：内容创作、修改文字、生成标题。如果内容还没定稿，请先用 `media-content-writer` 完成撰写。

---

## 文件结构

```
media-formatter/
├── SKILL.md                      ← Agent 执行指令主文件
├── README.md                     ← 本文档（供人阅读）
└── references/
    └── formatting-guide.md       ← 完整排版规范（字号/颜色/间距/所有组件 HTML 代码）
```

## 品牌视觉规范速览

| 参数 | 值 |
|------|----|
| 字体 | PingFang SC |
| 正文字号 | 15px |
| 正文字色 | #222222 |
| 品牌色 | #0251FF |
| 背景色 | #FFFFFF |
| 行间距 | 2 |

**标题体系**：一级大标题（24px 居中）→ 二级章节标题（品牌蓝斜体序号 + 20px 标题）→ 三级步骤标题（STEP 实心胶囊 + 17px 标题）

**可用组件**：蓝色信息卡、暖色提醒卡、默认卡片、引用块、对比栏、核心数据卡、效率对比条、投票栏、答案揭晓栏、分割线

## 如何使用

### 标准流程

1. 在 `media-content-writer` 中完成内容撰写和多轮修改
2. 确认内容定稿、标题选定
3. 调用本 Skill，将定稿正文粘贴给 Agent

```
「内容写好了，帮我排版」
「可以排版了，标题用第二个，正文如下：[粘贴正文]」
「帮我把这篇文章生成公众号 HTML」
```

### Agent 需要你提供

- ✅ 已定稿的文章正文（纯文字）
- ✅ 选定的标题
- ✅ 副标题（如有）
- ✅ 配图占位标注是否保留（默认保留）

## 每次输出包含

```
排版说明     ← 使用的标题、组件清单、配图占位数量
公众号 HTML  ← 从 <section> 到署名行的完整代码，可直接粘贴
```

---

## 标准文末模块

每篇文章排版后会自动追加，无需手动添加：

- 私信回复关键词资料包引导（Figma规范表 / AI动效库 / 动效配置指令集）
- 品牌介绍段落（Designing for Business Impact）
- 商务合作留言引导
- 署名行（12px / 斜体 / #b2b2b2）

---

## 如何粘贴到公众号

1. 复制 Agent 输出的完整 HTML 代码
2. 打开微信公众号编辑后台，新建图文
3. 点击编辑器右上角「</>」（源代码模式）
4. 将 HTML 代码粘贴进去
5. 切回普通模式预览效果
6. 在配图占位处手动插入对应图片

---

## 排版规范维护

所有样式参数和组件代码集中在 `references/formatting-guide.md`。

如需更新品牌色、调整字号、新增组件，只需修改该文件，无需改动 SKILL.md。修改后的规范会在下次调用时自动生效。

**禁止事项**（排版铁律）：
- 不引入渐变色块、阴影
- 不使用圆角大于 12px 的元素
- 不在三类标签（斜体序号 / STEP胶囊 / 空心方框）之间混用
- 不修改正文任何文字内容
